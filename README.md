# nbiot-tool-mobile

本工具主要针对NB-Iot相关测试开发工具的辅助，帮助工程师快速处理事务，减少加班时间。

现已开发完成功率转换，频率换算，后续会增加其他工具。

> 项目使用Vue开发，如果你有想法或工具代码，欢迎添砖加瓦。
